# Todo Obsidian - PRD

## 프로젝트 개요
Obsidian의 미니멀하고 다크한 디자인 감성을 가진 고급 투두(할 일) 관리 앱

## 핵심 기능 (MVP)
1. **할 일 관리**: 추가, 완료 토글, 삭제
2. **카테고리 분류**: 카테고리별 할 일 관리
3. **우선순위**: 높음/보통/낮음 3단계
4. **검색 및 필터**: 키워드 검색, 상태/카테고리 필터
5. **다크 테마**: Obsidian 스타일 다크 UI
6. **로컬 저장**: LocalStorage 기반 데이터 영속성
7. **통계 대시보드**: 완료율, 카테고리별 현황

## 기술 스택
- 순수 HTML + CSS + JavaScript (단일 파일)
- LocalStorage (데이터 저장)
- 외부 의존성 없음

## 디자인 컨셉
- Obsidian의 다크 퍼플/슬레이트 그라데이션
- 글래스모피즘 카드 UI
- 부드러운 애니메이션과 트랜지션
- 반응형 (모바일 지원)
