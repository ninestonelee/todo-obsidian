# 📒 Todo Obsidian

이 볼트(Vault)는 **Todo Obsidian App**과 연동하여 할 일을 관리하기 위한 공간입니다.

## 🚀 시작하기

1. **앱 실행**: 이 폴더 내의 `index.html` 파일을 브라우저로 엽니다.
2. **할 일 저장**: 앱에서 관리하는 할 일은 브라우저의 `LocalStorage`에 저장됩니다.
3. **노트 작성**: `todos/` 폴더 내에 마크다운 파일로 상세 내용을 작성하세요.

## 📂 폴더 구조

- **`.obsidian/`**: 옵시디언 설정 폴더 (테마, 폰트 등)
- **`todos/`**: 마크다운 할 일 일기/노트 저장 폴더
- **`index.html`**: 프리미엄 투두 관리 웹 앱
- **`prd.md`**: 프로젝트 기획 문서
