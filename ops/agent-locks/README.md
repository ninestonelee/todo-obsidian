# Agent Locks (작업 잠금 파일)

GIT_MULTI_AGENT_POLICY v2 섹션 5에 따른 작업 잠금 디렉터리.

## 사용법

작업 시작 시 `{에이전트명}.md` 파일을 생성하고, 완료 시 내용을 비우거나 삭제한다.

## 파일 형식

```markdown
| 항목 | 내용 |
|------|------|
| 작업 ID | #이슈번호 또는 작업명 |
| 수정 대상 | src/components/Todo.tsx, src/api/sync.ts |
| 시작 시각 | 2026-03-09 14:00 |
| 예상 종료 | 2026-03-09 16:00 |
| 브랜치 | claude/add-search-filter |
```

## 경합 규칙

동일 파일을 둘 이상의 에이전트가 수정하려 할 때, **먼저 잠금을 건 에이전트가 우선**한다.
