# Todo Obsidian 사용자 매뉴얼 및 초기 설치 가이드

Todo Obsidian은 옵시디언 마크다운 노트와 실시간 양방향으로 동기화되는 아름답고 직관적인 웹 할 일 관리 애플리케이션입니다.

---

## 🚀 초기 설치 및 연동 가이드

앱을 옵시디언과 연결하여 마법처럼 동기화되는 경험을 하려면 다음 3단계 설정이 필요합니다.

### 1단계: 옵시디언 플러그인 설정
옵시디언 내의 데이터를 웹 앱에서 읽고 쓰기 위해 **Local REST API** 플러그인이 필요합니다.

1. 옵시디언의 **설정 ⚙️ > 커뮤니티 플러그인 > 탐색** 메뉴로 이동합니다.
2. `Local REST API`를 검색하여 설치하고, **활성화(Enable)** 합니다.
3. 활성화된 플러그인의 **옵션(Options)** 에 들어갑니다.
4. 다음 포트 및 보안 설정을 정확히 맞춥니다:
    * **Port**: `27124` (앱과 통신하기 위한 전용 포트)
    * **HTTPS (Secure)**: **On** (보안 통신 필수 켬)
5. **API Key (Bearer Token)** 항목의 복잡한 문자열을 전체 복사해둡니다. (이 키는 앱과 통신할 때 인증 수단으로 사용됩니다)

### 2단계: 브라우저 보안 인증 허용
옵시디언 플러그인은 일종의 소규모 개인 서버 역할을 합니다. 처음 접속할 때 브라우저에서 '안전하지 않은 연결'로 차단할 수 있습니다.

1. 크롬, 엣지, 사파리 등 웹 브라우저를 엽니다.
2. 주소창에 `https://127.0.0.1:27124` 를 직접 입력하고 엔터를 칩니다.
3. "연결이 비공개로 설정되어 있지 않습니다" (크롬 기준) 경고창이 뜹니다.
4. 하단의 **[고급 설정]**을 누르고, **[127.0.0.1(안전하지 않음)으로 이동]** 글씨를 클릭합니다.
5. (빈 화면이나 JSON 코드가 뜬다면 정상적으로 보안 차단이 해제된 것입니다!)

### 3단계: Todo Obsidian 앱 설정
이제 앱에 옵시디언과 연결할 문(Port)과 열쇠(API Key)를 알려주면 됩니다.

1. `index.html` 파일을 더블클릭하여 브라우저에서 엽니다.
2. 앱 화면 우측 하단의 **⚙️ 설정(톱니바퀴) 아이콘**을 클릭합니다.
3. 설정 창에 다음 정보를 입력합니다:
    * **API 포트 번호**: `27124` (수정 불필요)
    * **API Key (Bearer Token)**: 1단계에서 복사해둔 긴 텍스트를 붙여넣습니다.
    * **타겟 마크다운 파일 경로**: 동기화할 노트의 위치를 적습니다. (예: `Todos.md` 또는 `Tasks/Daily.md`)
4. **[저장]** 버튼을 누릅니다.

🎉 **연결 완료!**
화면 하단에 "옵시디언 데이터를 동기화했습니다" 토스트 메시지가 뜨면 성공입니다! 이제 화면에서 할 일을 추가하거나 수정하면, 옵시디언의 해당 마크다운 파일 내용도 즉시 업데이트됩니다.

## 💡 주요 기능 사용법

### 1. 할 일 추가하기
* 상담 네모 상자에 "이메일 답장하기" 처럼 내용을 적고 엔터를 치거나 `+` 버튼을 누릅니다.
* 카테고리(업무, 개인 등)와 중요도(불꽃🔥, 번개⚡, 수면💤아이콘)를 미리 선택해둘 수 있습니다. 옵시디언에는 해시태그(`#업무`)와 기호(`🔥`)로 변환되어 저장됩니다.

### 2. 상태 필터링 및 카테고리
* 앱 최상단의 `전체 | 진행중 | 완료` 칩(알약 모양 버튼)을 클릭하여 상태별 모아보기가 가능합니다.
* 바로 아래의 탭 메뉴를 통해 카테고리별(업무, 개인 등)로 할 일을 분류해서 볼 수 있습니다. 오른쪽의 `+ 카테고리` 버튼으로 언제든 새로운 항목을 만들 수 있습니다.

### 3. 상태 변경 및 삭제
* **완료/취소**: 할 일 왼쪽의 체크박스를 클릭하세요.
* **삭제**: 할 일 오른쪽의 모양 버튼을 누르면 완전히 지워집니다.
* 앱에서 누른 액션은 옵시디언 파일 원본에도 즉각 반영됩니다.

### 4. 바로가기 팁 🪄
할 일 카드 우측의 **상자탈출(Open in new) 아이콘**을 클릭해보세요. 컴퓨터에 켜져 있는 `옵시디언` 앱이 자동으로 화면 위로 호출되면서, 해당 할 일이 담겨있는 원본 마크다운 파일을 즉시 띄워줍니다. (앱 이름이 `todo-obsidian`인 환경 최적화)
